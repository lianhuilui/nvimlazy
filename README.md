# Keymappings Summary

## File Operations
| Keymapping         | Description                        |
|--------------------|------------------------------------|
| `<leader>e`        | File Explorer                      |
| `<leader>sf`       | Find files                         |
| `<leader>sF`       | Find files (only in git)           |
| `<leader><space>`  | Recent Files                       |
